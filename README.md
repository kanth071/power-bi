Description:
This Power BI dashboard analyzes global malnutrition using key indicators such as stunting, wasting, underweight, and overweight. The dashboard provides a clear visual comparison across countries and income classifications to understand nutrition disparities worldwide.

Objective:
To identify patterns and trends in malnutrition and support data-driven decision-making for public health and policy planning.

Tools Used:
- Power BI Desktop
- Excel / CSV Dataset
- DAX for calculations

Key Features:
- KPI cards for quick insights
- Income classification analysis
- Country-level underweight trends
- Clean and interactive visual design

Conclusion:
The dashboard highlights that low-income countries experience higher levels of undernutrition, while middle-income countries show a dual burden of malnutrition.
